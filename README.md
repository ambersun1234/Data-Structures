# Data Structures
Collection of all data structures implementation with c++

## Clone repo
```=1
git clone https://github.com/ambersun1234/Data-Structures
```

## Run
```=1
make compile stack
make run stack
```

## Run test
```=1
make unit stack
make test stack
```
